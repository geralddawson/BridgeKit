## ⬇️ Download BridgeKit

👉 https://github.com/geralddawson/BridgeKit/releases/latest

---

### How to install

1. Download the ZIP file  
2. Double-click to extract  
3. Move **BridgeKit.app** to Applications  
4. Right-click → Open (first time only)

---

### macOS warning

If macOS blocks the app:
- Go to **System Settings → Privacy & Security**
- Click **Open Anyway**
